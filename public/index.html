Project Overview

Summary

The aim of this project is to leverage data storytelling techniques by applying acquired skills and knowledge to real-world scenarios. By integrating amCharts and SVG (Scalable Vector Graphics) for creating interactive data visualizations within the Express.js framework, this project aims to deliver engaging and insightful experiences to website visitors.

This interactive nature encourages audience engagement, enabling them to uncover hidden patterns, trends and insights that might otherwise remain obscured in raw data.

The topic of this project is to discover the insight of Hong Kong Waste Disposal. The analysis of statisitics would discuss in following part.

Objectives

Objective 1: To utilize Data visualizations tool (SVG, amcharts)

Objective 2: To apply javascript to manipuate data

Objective 3: To discover the insight of Hong Kong Waste Disposal

Objective 4: To design a great interactive interface

Methodology
  1. Import necessary CSS and JavaScript libraries, such as Bootstrap, amCharts, and D3.js

  2. Develop a page structure consists of a navigation bar and a content section by using Bootstrap's grid system

  3. Visualize the data by using amCharts library and D3.js library
